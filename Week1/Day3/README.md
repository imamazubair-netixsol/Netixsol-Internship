# Video Game Sales Dataset Analysis Using Pandas

## Project Overview

This project demonstrates the use of Pandas for data exploration, filtering, selection, grouping, and handling missing values. The analysis is performed on a video game sales dataset and focuses on answering business-oriented questions using only Pandas operations without any data visualization.

The objective is to strengthen practical data manipulation skills that are essential for Data Analysis and Data Science roles.

---

## Dataset

**Dataset:** Video Game Sales Dataset
**Source:** [Kaggle — Video Game Sales](https://www.kaggle.com/datasets/gregorut/videogamesales)

This dataset contains a list of video games with sales greater than 100,000 copies. It was generated by scraping [vgchartz.com](http://vgchartz.com/) using BeautifulSoup in Python. The scraping script is available at [github.com/GregorUT/vgchartzScrape](https://github.com/GregorUT/vgchartzScrape).

There are **16,598 records** across **11 columns**.

| Column | Description |
|---|---|
| `Rank` | Ranking of overall sales |
| `Name` | Name of the game |
| `Platform` | Platform of release (e.g. PC, PS4) |
| `Year` | Year of release |
| `Genre` | Game genre |
| `Publisher` | Publisher of the game |
| `NA_Sales` | Sales in North America (millions) |
| `EU_Sales` | Sales in Europe (millions) |
| `JP_Sales` | Sales in Japan (millions) |
| `Other_Sales` | Sales in the rest of the world (millions) |
| `Global_Sales` | Total worldwide sales (millions) |

## Data Cleaning Steps

Before analysis, the following cleaning steps were applied on a safe copy of the original DataFrame:

- `Year` column converted from `float64` → `Int64`
- Missing `Publisher` values (58) filled with `"Unknown"`
- Missing `Year` values (271) filled with the **median year**
- No duplicate records found

## Business Questions Answered

The notebook answers the following 8 questions using Pandas filtering and selection techniques:

1. Which 10 games generated the highest global sales?
2. Which platform generated the most total global sales?
3. Which game genre is most profitable globally?
4. Which publisher has sold the most games worldwide?
5. Which games sold more than 10 million copies globally?
6. How many games were released for the PlayStation 2 (PS2) platform?
7. What are the top 5 games where EU sales were higher than NA sales?
8. What is the average global sale for Action games vs Strategy games?

## Key Pandas Operations Used

**Sorting & Filtering**
```python
df.sort_values(by="Global_Sales", ascending=False)
df[df["Global_Sales"] > 10]
df[df["EU_Sales"] > df["NA_Sales"]]
```

**Grouping & Aggregation**
```python
df.groupby("Platform")["Global_Sales"].sum()
df.groupby("Genre")["Global_Sales"].mean()
```

**Missing Value Handling**
```python
df.isnull().sum()
df["Publisher"].fillna("Unknown")
df["Year"].fillna(df["Year"].median())
```

**Type Casting**
```python
df["Year"].astype("Int64")
```

**Safe Copying**
```python
cleaned_df = df.copy()
```

## Sample Insights

- **Wii Sports** is the highest-selling game globally with **82.74 million** copies sold.
- **PS2** is the highest-grossing platform of all time in this dataset with over **1,255 million** in total sales.
- **Action** is the most profitable genre, generating over **1,751 million** in global sales.
- **Nintendo** dominates publisher sales with nearly **1,787 million** in total global revenue.
- Only a small number of titles cross the **10 million sales** mark.
- In some titles like *Nintendogs* and *GTA V*, EU sales actually outpaced NA sales.
- Action games average **0.53M** in global sales vs **0.26M** for Strategy games.
